# FraggyMan BMI Tool
A tool for calculating BMIs. Made for Web Browsers.

# How to use
Use the buttons at the top to select your unit. (Either Standard (US) or Metric.) Then, type in the height and weight and the app will automatically calculate the result. The result will be colored based on how high it is and a category will be assigned based on how high it is. (The category will be visible right next to the output.)
